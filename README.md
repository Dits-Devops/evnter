# EVNTER - Simple Event Ticketing Platform

EVNTER adalah platform tiket event yang sederhana dan user-friendly, terinspirasi dari Eventbrite dengan fokus pada kesederhanaan UI/UX untuk pengguna non-teknis.

## Fitur Utama

### User (Peserta)
- Browse dan cari event
- Beli tiket dengan satu klik
- Generate QR code otomatis
- Lihat tiket saya
- Bagikan ke WhatsApp

### Organizer (Pro User)
- Buat event
- Kelola peserta
- Scan QR untuk check-in
- Dashboard event

### Admin
- Kelola user dan role
- Approver pro user
- Statistik platform

## Tech Stack
- Next.js 15+ dengan TypeScript
- Tailwind CSS
- Supabase (PostgreSQL)
- QR Code (qrcode.react)
- JWT Authentication

## Bahasa
Seluruh UI menggunakan Bahasa Indonesia untuk kemudahan penggunaan.
